📧 EmailMaster

⚠️ Vercel - Frontend Apenas
Esta versão na Vercel contém apenas o frontend. As funcionalidades completas requerem o backend Node.js.
✅ Localhost Funcionando
Execute npm run dev localmente para acessar a versão completa em http://localhost:5000
🚀 Deploy Completo Recomendado
Para deploy completo com backend + banco de dados, use:
  • Railway: railway.app (~$5/mês)
  • Render: render.com (gratuito)
  • Heroku: heroku.com

🔧 Funcionalidades

  • ✅ Interface React moderna
  • ✅ Sistema de autenticação
  • ✅ Gerenciamento de usuários
  • ✅ Sistema de emails
  • ⚠️ Backend APIs (apenas localhost/Railway/Render)
  • ⚠️ Banco PostgreSQL (apenas localhost/Railway/Render)
  • ⚠️ SMTP Email (apenas localhost/Railway/Render)

Consulte o arquivo DEPLOYMENT.md no repositório para instruções completas de deployment.
